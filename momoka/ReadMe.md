## 命名规则
### 文件命名规则
如果一个文件里面仅仅是声明了一个类，那么这个文件名就要和里面的类同名

头文件放在include目录下，源文件放在src目录下

### 变量命名规则
目前为止在有关engine的实现上我使用了匈牙利命名法，毕竟这部分比较贴近底层，使用的也都是windows api

计划在之后的游戏逻辑开发部分采用一种不同的命名方式，到时候会补充这部分
